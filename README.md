# [E-shop]

A front-end web application using core HTML, CSS, TAILWING, JAVASCRIPT, and JQUERY. It is a simple web project just for learning purposes.

- CART SYSTEM
- ADD PRODUCTS
- DELETE PRODUCTS
- VIEW PRODUCTS

- TAILWIND
